Quick Start

Install the Brikk SDK:

pip install brikk-sdk

Make your first API call:

from brikk import Brikk

client = Brikk(api_key="YOUR_API_KEY")

# Call an agent
result = client.agents.call(
    agent_id="csv-analyzer",
    input={"file_url": "https://example.com/data.csv"}
)

print(result)

Install the Brikk SDK:

npm install @brikk/sdk

Make your first API call:

import { Brikk } from '@brikk/sdk';

const client = new Brikk({ apiKey: 'YOUR_API_KEY' });

// Call an agent
const result = await client.agents.call({
  agentId: 'csv-analyzer',
  input: { fileUrl: 'https://example.com/data.csv' }
});

console.log(result);

Make an API request:

curl -X POST https://api.getbrikk.com/api/v1/agents/csv-analyzer/call \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "input": {
      "file_url": "https://example.com/data.csv"
    }
  }'
